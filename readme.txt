
# Marketing Before Funding

## Installing

1. Get the repo going. 
2. Install jekyll following [this guide](https://jekyllrb.com/docs/installation/). You'll need up-to-date versions of Ruby, Ruby Gems and NodeJS. Homebrew helps too.
3. Install these gems: jekyll-paginate, redcarpet, jekyll-sitemap, compass.

## Usage
1. Run the site using [this guide](https://jekyllrb.com/docs/usage/).
2. Stop the site builds by pressing Ctrl + C.
3. Always do the right thing.
